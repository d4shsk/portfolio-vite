# Лабораторная работа №2
## Сборка сайта с помощью Node.js, Vite и SASS. 

### Технологии
* Vite
* Bootstrap 5
* SCSS
### Структура проекта
```text
my-portfolio/
├── public/               # Статические ресурсы (логотипы, фавиконы)
├── src/
│   ├── scss/
│   │   └── style.scss   # Стили и кастомные переменные SASS
│   └── main.js           # Основной файл логики (импорт стилей)
├── index.html            # Разметка страницы
├── package.json          # Описание проекта и список зависимостей
```

### Команды для старта

1.  `npm install` — обязательная команда после создания проекта для загрузки всех библиотек.
2.  `npm run dev` — запускает локальный сервер. Сайт будет доступен по адресу `http://localhost:5173`.

### Команда для сборки 
`npm run build` — преобразует SASS в CSS и собирает готовый сайт в папку **`dist`**.
